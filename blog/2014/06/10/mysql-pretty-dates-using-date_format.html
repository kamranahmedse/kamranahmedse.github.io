MySQL - Pretty dates using date_format()

I use dates on daily basis. For example in the ERP system that I have been working upon, dates are the most crucial part e.g. transactions, user logs, stock records, payments records etc. in fact each and every action that any user performs is governed by a date. So it wonâ€™t be wrong to say that, I use DATE and DATETIME datatypes as much as I use VARCHAR. The default format used by MySQL to store date and time is:

YYYY-MM-DD HH:MM:SS # e.g. 2014-06-10 06:11:00

Although this format works fine, I donâ€™t consider it fair to put the user into thinking over it. The date time should be such that the user understands it in a glimpse. To format the dates, I use this nice MySQL function to format dates called DATE_FORMAT(). Example below shows, how you may use it:

DATE_FORMAT(NOW(),'%M %e, %Y @ %h:%i %p')
